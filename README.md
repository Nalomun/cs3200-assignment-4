# cs3200-assignment-4
Query a database with SQL

Quinn Lambert (Nalomun)

This assignment is 9 SQL queries on a music store database (Chinook). Q1-2 are basic joins, Q3-4 group/filter customers by state, Q5-6 use pattern matching and multi-table joins to find artists by album titles or playlist/media type, Q7 counts artists with 10+ MPEG tracks, Q8 calculates playlist lengths in hours, and Q9 is a creative query using a window function across 3+ tables.
